## Primeros Pasos

### Instalacion

1. Descarga el APK desde la seccion de releases
2. Habilita "Instalar desde fuentes desconocidas" en tu Android
3. Instala el APK
4. Abre la app

### Registro / Login

1. **Con Google**: Toca "Continuar con Google" para login rapido
2. **Con Email**: Ingresa tu correo y contrasena

Al ingresar por primera vez, la app te guiara para crear tu primera cuenta.

## Dashboard

La pantalla principal muestra un resumen de tus finanzas:

### Elementos

- **Balance Total**: Suma de todas tus cuentas activas
- **Ingresos/Gastos del Mes**: Totales del periodo actual
- **Grafico de Balance**: Tendencia de los ultimos dias
- **Transacciones Recientes**: Ultimos movimientos
- **Acceso Rapido**: Botones a las secciones principales

### Navegacion

- **Barra inferior**: Dashboard, Cuentas, (+), Movimientos, Reportes
- **Boton (+)**: Agregar nueva transaccion
- **Menu lateral**: Acceso a todas las funciones

## Cuentas

Gestiona todas tus cuentas financieras.

### Tipos de Cuenta

| Tipo | Descripcion |
|------|-------------|
| Efectivo | Dinero en mano |
| Cuenta Bancaria | Cuenta de cheques/debito |
| Ahorros | Cuentas de ahorro, CDT |
| Tarjeta de Credito | Credito rotativo |
| Billetera Digital | Nequi, Daviplata, PayPal |
| Inversiones | Acciones, cripto, fondos |

### Crear Cuenta

1. Ve a la seccion **Cuentas**
2. Toca el boton **(+)**
3. Selecciona el tipo de cuenta
4. Ingresa:
   - Nombre (ej: "Banco Azteca")
   - Balance inicial
   - Moneda (MXN por defecto)
   - Color e icono (opcional)
5. Toca **Guardar**

### Editar/Eliminar

- **Editar**: Toca la cuenta > Editar
- **Eliminar**: Desliza a la izquierda > Eliminar

## Transacciones

Registra tus ingresos, gastos y transferencias.

### Tipos de Transaccion

1. **Ingreso**: Dinero que entra (salario, ventas, etc.)
2. **Gasto**: Dinero que sale (compras, pagos, etc.)
3. **Transferencia**: Mover dinero entre tus cuentas

### Registrar Transaccion

1. Toca el boton **(+)** central
2. Selecciona el tipo (Gasto/Ingreso/Transferencia)
3. Ingresa:
   - Monto
   - Cuenta
   - Categoria
   - Descripcion (opcional)
   - Fecha (por defecto: hoy)
4. Toca **Guardar**

### Categorias

Las categorias te ayudan a organizar y analizar gastos:

**Gastos**:
- Alimentacion, Transporte, Vivienda
- Servicios, Salud, Entretenimiento
- Ropa, Educacion, Mascotas, etc.

**Ingresos**:
- Salario, Freelance, Ventas
- Inversiones, Regalos, Otros

### Filtrar Transacciones

En la pantalla de Movimientos puedes filtrar por:
- Rango de fechas
- Tipo (ingreso/gasto)
- Cuenta
- Categoria

## Presupuestos

Define limites de gasto por categoria.

### Crear Presupuesto

1. Ve a **Presupuestos** (desde el menu lateral)
2. Toca **(+)**
3. Selecciona categoria
4. Define el monto limite
5. Selecciona periodo (semanal/mensual/anual)
6. Toca **Guardar**

### Seguimiento

- **Verde**: Menos del 80% gastado
- **Amarillo**: Entre 80-100% gastado
- **Rojo**: Presupuesto excedido

Recibes alertas cuando te acercas o superas el limite.

## Metas de Ahorro

Establece objetivos financieros y sigue tu progreso.

### Crear Meta

1. Ve a **Metas** (desde el menu lateral)
2. Toca **(+)**
3. Ingresa:
   - Nombre (ej: "Vacaciones 2026")
   - Monto objetivo
   - Fecha limite (opcional)
   - Icono y color
4. Toca **Guardar**

### Agregar Ahorro

1. Abre la meta
2. Toca **Agregar ahorro**
3. Ingresa el monto
4. Confirma

El progreso se muestra visualmente con barra de progreso.

## Asistente IA (Fina)

Fina es tu asistente financiera personal basada en inteligencia artificial.

### Como Usar

1. Abre el chat (icono en la esquina)
2. Escribe tu pregunta
3. Fina respondera con consejos personalizados

### Ejemplos de Preguntas

- "Como van mis finanzas este mes?"
- "En que estoy gastando mas?"
- "Dame consejos para ahorrar"
- "Estoy cumpliendo mis presupuestos?"
- "Cual es mi balance actual?"
- "Analiza mis gastos de la semana"

Fina usa TUS datos reales para dar respuestas personalizadas.

## Preguntas Frecuentes

### La app funciona sin internet?

Si! Finanzas Familiares usa tecnologia "offline-first". Todos tus datos se guardan localmente y se sincronizan cuando hay conexion.

### Como recupero mis datos si cambio de telefono?

1. Asegurate de tener sesion iniciada (Google o email)
2. En el nuevo telefono, inicia sesion con la misma cuenta
3. Tus datos se sincronizaran automaticamente

### Mis datos estan seguros?

- Los datos se encriptan localmente
- La sincronizacion usa HTTPS
- Cada usuario solo ve sus propios datos (Row Level Security)

### Puedo usar la app en computadora?

Si, hay version para Linux. Proximamente Windows y Web.

### Como elimino mi cuenta?

Configuracion > Datos > Eliminar cuenta. Esta accion es irreversible.

### La IA tiene acceso a mis datos bancarios reales?

No. Fina solo usa los datos que TU registras en la app. No se conecta a bancos.
